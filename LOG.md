# 2회차 (2026.05.18)

#오늘 한 일
만든 것
→ 가스 누출 위험도 시각화 시스템

작동 방법 
→ MQ2 센서가 가스 농도를 감지하면
   RGB LED가 색상으로 위험도를 표시

색상 의미
→ 초록색: 안전
→ 노랑색: 주의
→ 빨강 점멸: 위험


- MQ2 센서로 가스 감지
- RGB LED 3단계 효과
  - 안전: 초록↔파랑 파도 효과
  - 주의: 노랑 천천히 깜빡
  - 위험: 빨강 빠르게 점멸


쓸 수 있는 곳
→ 화학 실험실, 가정, 공장 등
   가스 누출 위험이 있는 곳

#안된것
   잘되는지 확인해보려고 피코를 켰는데 작동은 되는데 LED에 불이 안들어와서 다시 바꿔서 해봤는데 계속 불이 안들어왔다
   그래서 코드가 문제인가 싶어서 코드도 바꿨는데 계속 안됬다 그래서 다시 지피티한테 짜달라고 했는데 또 안들어왔다
   그래서 또 TIMING = (280, 515, 515, 745)
led = NeoPixel(Pin(16), 1, timing=TIMING) 이게 없길래 이걸 또 추가해서 했더니 그래도 안됬다

#잘된것 
   빨리 주제를 정하게 되서 코드 짜기 수월했다
   
   

#수정한 것

기존 임계값이 너무 높았어서 가만히 두어도, 입김을 불어도 계속 안전이라고만 떠서 코드를 120 이하 → 안전 120~150 → 주의, 150 이상 → 위험 으로 설정했다.

그리고 원래는 10개중 하나만 LED에 불빛이 나왔는데 모두 나오게 설정했다

안전할때 나오는 불빛이 너무 눈이 아파서 좀 잔잔하게 바꾸었다

어디까지가 좋고 나쁜지 바로 눈에 안보여서 플로터를 키면 좋음 보통 나쁨의 기준선이 표시되어있게해 한눈에 들어오게 수정했다
