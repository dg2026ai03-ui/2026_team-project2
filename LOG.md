# 팀원 : 수도여고 신연서, 최윤영

# 2회차 (2026.05.18)

#오늘 한 일
만든 것
→ 가스 누출 위험도 시각화 시스템

작동 방법 
→ MQ2 센서가 가스 농도를 감지하면
   RGB LED가 색상으로 위험도를 표시

색상 의미
→ 초록색: 안전
→ 노랑색: 주의
→ 빨강 점멸: 위험

심화과정
### Level 1 (기본)
- MQ2 센서로 가스 감지
- RGB LED 3단계 효과
  - 안전: 초록↔파랑 파도 효과
  - 주의: 노랑 천천히 깜빡
  - 위험: 빨강 빠르게 점멸

### Level 2 (통계 추가)
- 측정 횟수 기록
- 최대/최소/평균값 계산
- 주의/위험 발생 횟수 기록
- 10회마다 통계 자동 출력

### Level 3 (심화 완성)
- 센서 예열 기능 추가 (30초)
- 4단계 위험도 세분화
  - 안전/주의/위험/긴급
- 긴급 모드 LED 추가 (보라색)
- 연속 3회 위험 시 긴급 경보
- 최근 10회 히스토리 저장
