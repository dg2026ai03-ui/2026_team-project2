# 🔬 가스 누출 경보 위험도 시각화 시스템
- MQ2 센서로 가스 농도를 감지하고
  RGB LED 색상으로 위험 단계를 표시한다

## 어떤 부품으로 만들었나
- 라즈베리파이 피코
- MQ2 가스 센서
- RGB LED 1개

가스 없을 때
→ 🟢 초록↔파랑으로 파도처럼 일렁인다.

라이터 가스 살짝 대면
→ 🟡 노랗게 천천히 깜빡인다.

라이터 가스 많이 대면
→ 🔴 빨갛게 빠르게 점멸한다.




최근 3회 측정값을 비교해서
공기질이 어떻게 변하는지 예측

3번 전보다 지금이 높으면
→ 📈 공기질 나빠지는 중!

3번 전보다 지금이 낮으면
→ 📉 공기질 좋아지는 중!

비슷하면
→ ➡️ 공기질 유지 중!


#최고치 기록 (peak_ppm)
   → 역대 가장 높은 PPM 저장
   → 갱신 시 터미널에 출력

경고 횟수 카운터 (warn_count)
   → 안전 → 위험 될 때마다 +1
   → 누적 횟수 기록

측정 시간 (elapsed_time)
   → 시작부터 지금까지
   → 몇 분 몇 초 경과했는지
